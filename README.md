# 📚 Library Book Management System (FastAPI)

🚀 A complete backend application built using FastAPI that simulates a real-world Library Management System.  
This project demonstrates API design, validation, CRUD operations, and workflow handling.

---

## 📌 Project Overview

This system allows users to:
- Manage books
- Borrow and return books
- Handle waiting queues
- Perform advanced search and filtering
- Navigate large datasets using pagination

The project follows real-world backend development practices and covers all core FastAPI concepts.

---

## 🚀 Features

### 🔹 Core Features
- RESTful API development using FastAPI
- Pydantic validation for request data
- Structured API responses
- Error handling for invalid inputs

### 🔹 CRUD Operations
- Add new books
- View all books
- Update book details
- Delete books

### 🔹 Workflow System
- Borrow books
- Return books
- Queue system for unavailable books
- Automatic reassignment from queue

### 🔹 Advanced Features
- 🔍 Search books by keyword
- 🎯 Filter books (genre, author, availability)
- ↕️ Sort books (ascending / descending)
- 📄 Pagination support
- 🔗 Combined browse endpoint (filter + sort + pagination)
